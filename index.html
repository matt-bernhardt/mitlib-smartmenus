SmartMenus implementation

This is a more complex implementation of the jQuery SmartMenus plugin. It includes non-list markup in some flyout menus, including a `mega-menu` class approach that pushes flyouts to be a full-width rendering.

This was created by cribbing from this JSfiddle demonstration.
